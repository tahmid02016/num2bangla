# num2bangla
A Python module for converting number &amp; digits into Bangla words. For example '123' -> 'এক শত তেইশ'.

## Usage
```python
>>> print(num2bangla(123))
'এক শত তেইশ'
>>> print(num2bangla('123'))
'এক শত তেইশ'
>>> print(num2bangla(123.123))
'এক শত তেইশ দশমিক এক শত তেইশ'
>>> print(num2bangla('123.123'))
'এক শত তেইশ দশমিক এক শত তেইশ'
```
